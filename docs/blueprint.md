# **App Name**: BuildFlow Yido

## Core Features:

- AI Maintenance Diagnostic Tool: A generative AI-powered tool that allows users to describe building issues (cracks, leaks, etc.) and provides instant assessments of the likely cause and suggested repair procedures.
- Smart Quotation System: Enables users to submit request details and project parameters through a structured form to generate preliminary estimates.
- Service Portfolio CMS: A gallery view to showcase completed projects in categories like waterproofing, painting, and general repair, with high-quality imagery.
- Management Dashboard: An administrative interface for the business owner to track pending inquiries, project schedules, and client contact information.
- Unified Database Management: Reliable data storage for user-submitted quotes, client details, and portfolio content.
- Interactive Service Navigator: A categorized navigation menu with sub-pages detailing specialty areas: Waterproofing, Painting, and Facility Management.

## Style Guidelines:

- Primary color: Deep Integrity Blue (#214B8C) - evoking trust, expertise, and a sense of calm reliability for property maintenance.
- Background color: Misty Pearl (#F3F5F9) - a desaturated, bright background that emphasizes the structural and clean feel of modern construction.
- Accent color: Celestial Azure (#479EFF) - a vibrant, analogous hue to guide user actions and highlight diagnostic tool results.
- Font pairing: 'Space Grotesk' for technical headlines to give a blueprint, machined aesthetic; 'Inter' for body text to maintain superior readability and objective professionalism.
- Modular grid system based on architectural proportions, featuring high-contrast spacing and clearly defined sections to reflect order and stability.
- Smooth 'Reveal' entrance animations (vertical displacement and fade) to echo the slow and steady rising of a solid building project.
- Geometric line-icons with a technical weight, inspired by construction drafting, used to symbolize specific services and repair categories.
